# 当前主实验：几何恢复与 SAM3 same-ID memory writeback 因果消融。
#
# 固定 ScanNet++ 场景与 7 帧压力序列，暂不使用面积过大且首帧局部可见的
# bed(54)。cabinet(37) 和 wardrobe(68) 共享一次 StreamVGGT 几何提取，
# 但分别维护自己的 SAM3 obj_id、恢复 mask 和评估结果。
#
# 四个分支：
#   1. original
#   2. geometry_recovery_no_memory
#   3. geometry_recovery_same_id_memory
#   4. shuffled_geometry_same_id_memory
#
# 第 2、3 个分支使用逐像素完全相同的完整恢复 mask，唯一变量是是否写入
# SAM3 memory。第 4 个分支只打乱非 reference 帧的 StreamVGGT 几何对应，
# RGB、SAM3 全图文本候选和阈值保持不变。
#
# 帧 90 固定为 reference。若某个实例在帧 90 不可见，程序会直接报错，
# 不会静默使用未来 GT 帧初始化。

  --scene-id 00a231a370 \
  --instance-ids 37 68 \
  --frame-indices 90 105 119 130 140 210 240 \
  --reference-sequence-index 0 \
  --sam3-device cuda:3 \
  --geometry-device cuda:1 \
